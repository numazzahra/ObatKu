# ObatKu

An Android application developed to help users manage medication schedules through reminder notifications.

## Overview

ObatKu allows users to organize medication schedules, receive reminder notifications, and manage daily medication information through a simple and accessible mobile interface.

## Key Features

- Medication management
- Reminder scheduling
- Notification system
- Simple and intuitive interface

## Tech Stack

| Category | Technology |
|----------|------------|
| Language | Java |
| IDE | Android Studio |
| UI | XML |
| Database | SQLite |

## Installation

```bash
git clone https://github.com/numazzahra/ObatKu.git
```

Open the project using Android Studio and run it on an emulator or Android device.
